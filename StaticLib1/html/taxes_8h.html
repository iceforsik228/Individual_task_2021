Подробное описание

Файл h, который отвечает за объявление структуры и переменных для хранения информации из файла taxes.txt.

Автор
Чихичин Cтанислав Сергеевич
Версия
1.0
Дата
10.01.2022

Данный файл содержит структуру resource, в которой содержится информация о наименовании каждого ресурса, месячном тарифе за данный ресурс и зарезервированном значении k. Также файл содержит объявление глобальной переменной Q, которая содержит информацию о количестве ресурсов.
